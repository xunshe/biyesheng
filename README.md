### 运行环境 
```
JDK版本：1.7
IDE工具：eclipse
数据库: mysql
编程语言: Java
tomcat:  7.0
详细技术：HTML+CSS+JS+JSP+JAVA+JSP+MYSQL
```

### 主要功能 
毕业生求职招聘系统，本系统适用于毕业设计和课程设计，系统主要分为毕业生，招聘企业，管理员三个角色，其中用户毕业生功能主要有，登录，注册，查看公司招聘信息，个人基本信息，求职意向，联系方式，查看招聘信息，招聘企业主要功能有：管理招聘信息，注册企业用户，企业职位信息管理，管理员主要功能有：企业管理，毕业生管理，网站管理，个人管理
